# computational-linguistics-project
Here I have submitted my work about language prediction, in human and machine context.

The scripts show how classical NLP tools such as n‑gram language models and vocabulary growth curves can be linked to ideas from cognitive science and neurolinguistics. All scripts are deterministic and use only small toy examples.

**Repository structure**

example1_bigram.py – simple bigram language model with Laplace smoothing and perplexity

example2_surprisal.py – computing word‑level surprisal in a bigram model

example3_heaps_law.py – script illustrating Heaps’s law for vocabulary growth.

korpus.txt – example text file used by the example3, of course, its to small to illustrate it perfetlcy but I assume that this is enough for pedagogical purpouses.
